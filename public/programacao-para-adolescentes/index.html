Programação para adolescentes

Já tinha ouvido falar sobre o ensino de programação de computadores em escolas de ensino fundamental e médio ser uma preocupação importante para muitas pessoas.

Uma destas pessoas é Emmanuel Schanzer, depois de trabalhar como desenvolvedor de software em algumas empresas trocou de carreira para virar professor de matemática em escolas públicas de Boston, EUA.

Então, criou um trabalho no qual estudantes são estimulados a aplicar conceitos de matemática na programação do seu próprio jogo de computador. Há inclusive uma camiseta legal escrito “Eu programo meu próprio Vídeo-Game”.

O projeto se chama Bootstrap e pelo que pude ver é financiado pelo Google. Todo o material é público e disponibilizado no site.

Outro projeto semelhante é o Program By Design, com um currículo que vai de uma linguagem educacional até o Java.

Fiquei impressionado. E aproveito pra linkar duas iniciativas brasileiras muito parecidas, a primeira é o projeto novos talentos que buscou e treinou desenvolvedores de jogos em Florianópolis, SC, a segunda é o CDI, organização sólida, com um longo e belo trabalho de democratização da Informática.

Tem bastante gente buscando o potencial da informática para educar. Belo trabalho.
